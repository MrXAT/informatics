    Утилита ping - это очень простой инструмент для диагностики сети.
Она позволяет проверить доступен удаленный хост или нет и все.
Для этого утилита проверяет, может ли хост отвечать на сетевые
запросы с помощью протокола ICMP.
Этот очень простой инструмент можно применять для тестирования
неполадок сети и их исправления.
    Для проверки работоспособности сети часто
используется программа ping любого сайта, который всегда доступен,
например, google.com
